#Intallation
python -m pip install Django

#Verifying installation
django-admin --version

#Creating new website
django-admin startproject [Project name i.e demoWebsite]

#Starting project server
python demoWebsite\manage.py runserver

#Creating new app
python manage.py startapp [app name i.e music]

#Views
for different apps, we should create separate "urls.py" to handle the use requests for that specific app.
For example, to handle the requests for "music"app we need to create "urls.py" under the "music" app directory.
